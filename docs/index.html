Codebot API Documentation

Welcome to the Codebot API. This reference describes every command available for programming your robot inside the maze.

You write programs in the code editor using a sequence of commands. The robot executes them one step at a time — moving through the grid, collecting coins, and trying to reach the goal tile. Use move commands to control direction, and repeat blocks to avoid writing the same commands over and over.

Keywords are case-insensitive: moveUp, MOVEUP, and moveup are all identical. Whitespace (spaces, tabs, blank lines) is ignored, so indent freely to make your code readable.

All Commands
Type Command & Description
void moveUp() Moves the robot one cell upward.
void moveDown() Moves the robot one cell downward.
void moveLeft() Moves the robot one cell to the left.
void moveRight() Moves the robot one cell to the right.
block repeat(count) … end Repeats enclosed commands a fixed number of times (1–100).
