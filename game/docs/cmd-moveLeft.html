Overview > moveLeft
Command Detail
voidmoveLeft()

Moves the robot one cell to the left on the maze grid. If there is a wall directly to the left of the robot, this command has no effect and the robot stays in place.

Each moveLeft costs exactly one game frame.

Examples
Move left once

Advance the robot one cell to the left.

moveLeft
Move left several columns

Chain multiple moveLeft commands to travel further.

moveLeft
moveLeft
moveLeft
Backtrack after moving right

Navigate a corridor and then return.

moveRight
moveRight
moveDown
moveLeft
moveLeft
Use inside a repeat block

Use repeat to move left multiple columns concisely.

repeat(6)
    moveLeft
end
