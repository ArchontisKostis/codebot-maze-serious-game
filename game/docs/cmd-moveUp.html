Overview > moveUp
Command Detail
voidmoveUp()

Moves the robot one cell upward on the maze grid. If there is a wall directly above the robot, this command has no effect and the robot stays in place.

Each moveUp costs exactly one game frame.

Examples
Move up once

The simplest use: advance the robot one cell upward.

moveUp
Move up several times in a row

Chain multiple moveUp commands to travel further.

moveUp
moveUp
moveUp
Combine with other commands

Mix directions to navigate around walls.

moveUp
moveRight
moveUp
moveUp
moveLeft
Use inside a repeat block

Use repeat to avoid repeating the same command manually.

repeat(4)
    moveUp
end
