codebot.robot

Robot

codebot.robot
└─ Robot

The Robot is the programmable agent you control inside the maze. You give it a sequence of commands and it executes them one step at a time, moving through the grid until it reaches the goal — or runs out of instructions.

Programs are written as plain text. Commands are separated by whitespace (spaces, tabs, or newlines). Keywords are case-insensitive: moveUp, MOVEUP, and moveup are all valid.

Since:
1.0
Command Summary
Type Command & Description
