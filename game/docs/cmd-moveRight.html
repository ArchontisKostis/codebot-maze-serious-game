Overview > moveRight
Command Detail
voidmoveRight()

Moves the robot one cell to the right on the maze grid. If there is a wall directly to the right of the robot, this command has no effect and the robot stays in place.

Each moveRight costs exactly one game frame.

Examples
Move right once

Advance the robot one cell to the right.

moveRight
Move right several columns

Chain multiple moveRight commands to travel further.

moveRight
moveRight
moveRight
Navigate a corridor

Move right across a row, then turn and descend.

moveRight

Use inside a repeat block

Use repeat to move right multiple columns concisely.

repeat(8)
    moveRight
