Overview > moveDown
Command Detail
voidmoveDown()

Moves the robot one cell downward on the maze grid. If there is a wall directly below the robot, this command has no effect and the robot stays in place.

Each moveDown costs exactly one game frame.

Examples
Move down once

Advance the robot one cell downward.

moveDown
Descend multiple rows

Chain multiple moveDown commands to travel further.

moveDown
moveDown
moveDown
Navigate around a wall below

If a wall blocks direct descent, go around it.

moveRight
moveDown
moveDown
moveLeft
Use inside a repeat block

Use repeat to descend multiple rows concisely.

repeat(5)
    moveDown
end
